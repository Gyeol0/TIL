# Python 기초

## Syntax

### 주석(Comment)

* `#`한 줄 주석

* 여러 줄 주석

  1. 한 줄씩 `#`사용
  2. `'''`또는 `"""` 사용

* ```python
  # 한 줄 주석
  '''여러 줄
  주석
  여러 줄
  '''
  ```

### 코드 라인

* 파이썬에서는 `;`을 작성하지 않는다.

* 한 줄로 표기할 때는 `;` 작성이 가능하다.

* 출력

  * print(변수)
  * print('문자')

* 코드 1줄에 1문장을 원칙으로 한다.

  * ```python
    print('hello')print('hi')
    ```

    * 에러(invalid syntax)

  * ```python
    print('hello');print('hi')
    ```

    * 출력

## 자료 구조

### 순서에 따른 자료 구조

* 순서가 있는 자료 구조
  1. 리스트
     * `[]` 형태 또는 list()로 생성
     * 인덱스로 접근
     * 값 변경이 가능하다.
  2. 문자열
     * 저장된 그대로 순서가 유지된다.
     * 인덱스로 접근과 슬라이싱이 가능
     * 슬라이싱한 부분에 대한 변경이 불가능하다.
  3. range
     * range(범위)
     * 범위에 따라 seq(1,2,3...)과 같이 range가 생성
     * 지정한 범위와 순서에 따라 생성
* 순서가 없는 자료 구조
  1. 튜플
     * `()` 형태 또는 `tuple()`로 생성
     * 인덱스로 접근, 리스트와 비슷
     * 값 변경이 불가능하다. 리스트와 차이점
  2. set
     * 집합과 동일하게 처리하며 `set()` 또는 `{}`로 생성
     * 순서가 없다.
     * 자동으로 중복을 제거한다.
  3. 딕셔너리
     * 이름과 같이 사전이다.
     * 키 - 값 쌍으로 이루어져 있다.
     * `dict[Key]`로 접근이 가능하고 Value를 반환한다.
     * Key로 접근하기 때문에 순서와 상관이 없으며, 중복이 불가능하다.
     * Key로 접근하여 Value를 변경할 수 있다.
     * `dict.items()`를 사용하면 (Key, Value)로 묶인 튜플들로 구성된 `dict_items`(list와 비슷?) 반환

# 제어문

## 조건문

### if

* 참인지 거짓인지에 대한 조건식에 따라 수행한다.

* ```python
  if '조건식':
      '수행할 문장'
  elif '조건식':
      '수행할 문장'
  else:
      '수헹할 문장'
  ```

* `if`와 `:`를 사용하고 `들여쓰기`로 구분한다.

* `if`문 안에 `if`문을 사용할 수 있다.

* 조건식이 `True`이면 이에 대한 문장을 수행한다.

* `if` 조건이 `False`이면 `elif`로 넘어가고, 이도 `False`이면 `else`로 넘어 간다.

* `elif`개수에 대한 제한은 없다.

## 반복문

### for

* 정해진 횟수만큼 문장을 반복한다.

* ```python
  for x in range(1,10):
      '수행할 문장'
  ```

* `range`를 사용하면 범위 값을 x로 하여 반복한다.

  * range(1,10)이면 x에 1,2,3,4....9를 차례로 대입하여 문장을 수행한다.

* `range`자리에 `list` `tuple` `문자열`과 같은 데이터를 적용할 수 있다.

* `if`문을 중첩하여 사용할 수 있다.

* 수행할 문장에 `break`를 쓰면 반복문 끝나지 않아도 종료된다. 

* 수행할 문장에 `continue`를 쓰면 `continue`뒤의 문장을 수행하지 않고 다음 순번으로 넘어간다.

* 조건식에서 수행할 문장에 `pass`를 사용할 때가 있다.

  * 문법적으로 자리를 채울 때
  * 참, 거짓에 따라 아무 일도 수행하지 않는다.
  * `continue`는 반복문을 다음 순번으로 넘기지만, `pass`는 조건식을 넘어가지만 반복문 안에 있는 다음 문장은 그대로 실행한다.

* `break` `continue`를 통해 원하는 시점에 반복을 제어할 수 있다.

### while

* 조건식이 참일 경우 문장을 수행한다.

* for 문과 동일하게 `if` 문을 중첩하여 사용할 수 있다.

* ```python
  while '조건식':
      '수행할 문장'
  ```

* 조건식이 계속 참이면 무한대로 반복하며 무한 루프에 빠지게 된다.

* 무한 루프가 돌지 않도록 조건식과 반복 제어를 조정해야 한다.

* `break`를 수행하거나 조건식이 `False`가 되면 반복문이 중지된다.

* `break` `continue` `pass`는 for 문과 동일하게 사용된다.
